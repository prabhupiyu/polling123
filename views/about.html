Polling Application

This web application will be useful in gathering the public response of various topics via polls created. The result will be based on the voting done by users.

User can also create their own polls as well as vote for polls created by others. The polls can be sorted based on the categories they belong so as to facilitate the search of polls on various topics.

User can view the poll result either in bar chart or pie chart format based on its own choice
